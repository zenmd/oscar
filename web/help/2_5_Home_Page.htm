2.5           Home Page

 

QuatroShelter is organized into modules. Each module is represented in the Home Page by an icon and a link. When the user logs into QuatroShelter, the Home page is displayed. From this page the user may navigate to other screens in the system by using the links provided. Based on the security access of the user, some options may be disabled.

The following charts describe the various navigation options available in the Home Page.

       
     
   
 
 

 

 


Figure 3: Home Page

REPORTS

By selecting Reports the user can access the Reports function and the reports available to the user.

By selecting Help the user can access the help file of QuatroShelter.

G

By selecting Logout the user can safely close the application.

H

Using Quick Links enables the user to quickly access functions commonly used. Quick Links are found on the top row of the Main Bar

MENU BAR

Using Menu Bar, the user can access Modules or Functions within a Module directly, without having to return the main menu in the Home Page.

J

LINKS TO MODULES

By selecting any of the various Icons on your home Page.  A user can quickly link to the module selected.  Each module is represented by a different Icon and Link.
