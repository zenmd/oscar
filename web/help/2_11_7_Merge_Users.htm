2.11.7  Merge Users

 

The Merge Client functionality in QuatroShelter is used to deal with situations where more than one client record was created for a single individual. It may be decided to merge both client records into one. This function is the tool for this job.

 

 
   
 


 

 

           
           
 

 

 

 


Label

Option

Action

A

Client No.

Enter a Client No. to search for clients.

B

First Name

Enter a client’s Fast Name on this field to search for clients.

C

Last Name

Enter a client’s First Name on this field to search for clients.

D

Date of Birth

Enter a client’s birth date to search for clients.

E

Active?

Select Any to search from all clients, select Yes to select from active clients and select No to select for inactive clients.

F

Gender

Select Any to search from all clients or select specific gender to search clients from.

G

Client list

List of clients that match the client search based on the filter entered.

H

Selection checkboxes

Check boxes to select which clients are wanted merged.

I

Main Record Indicator

The button is used to select the main client record.

J

Close

Press the Close button to exit the Merge function.

K

Merge

Press Merge button to merge the selected clients.

L

Reset

Press the Reset button to clear the search filter.

M

Search Merged Records

Press the Search Merged Records button to search and list records that have already been merged.

N

Search

Press the Search button to search clients based on filter selection.
